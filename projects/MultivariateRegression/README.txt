Project Title: Multivariate Polynomial Regression

=====


Included one Python Notebook file:

- MultivariatePolynomialRegression.ipynb



Included the corresponding python file:

- MultivariatePolynomialRegression.py




===== To Open/Run Jupyter Notebook file =====


To open jupyter notebook file (.ipynb), open a terminal, go to the directory where the file is located, and run the following command:

$ jupyter notebook MultivariatePolynomialRegression.ipynb


Or, to open file in jupyter lab, run the following command:

$ jupyter lab MultivariatePolynomialRegression.ipynb




Use "Run" button in the notebook interface to run the above.





===== To Execute .py file =====


To run the Python file (.py), open a terminal, go to the directory where the file is located, and run the following command:

$ python3 MultivariatePolynomialRegression.py



At runtime, it will generate and save all the 4 plot files as .jpg in the same directory. These are:
- Figure_Plot1.jpg
- Figure_Plot2.jpg
- Figure_Plot3.jpg
- Figure_Plot4.jpg



===== Description =====

File MultivariatePolynomialRegression.ipynb contains:

- Loading the Excel dataset (contains 300 rows/data points).

- Cleaning dataset (removing outliers).

- Bivariate Polynomial Regression surface-fitting to the given set of data points in 3D.

- 3D plots, Surface-fit plots.
